# 📊 Student Social Media Addiction Analysis Dashboard

## 📌 Project Overview

The Student Social Media Addiction Analysis Dashboard is a Power BI project designed to analyze how social media usage impacts students' behavior, mental health, academic levels, and personal relationships. The project transforms raw student data into meaningful insights through interactive dashboards, helping identify addiction patterns and behavioral trends.

---

## 🎯 Project Objectives

* Analyze students' social media addiction levels.
* Understand the relationship between addiction scores and mental health scores.
* Compare addiction patterns across different academic levels.
* Examine conflicts caused by social media based on relationship status.
* Provide detailed student-level insights through drill-through analysis.

---

## 🛠️ Tools & Technologies Used

* **Power BI** – Dashboard creation and interactive data visualization.
* **Power Query** – Data cleaning, transformation, and preparation.
* **DAX (Data Analysis Expressions)** – Creating measures and analytical calculations.
* **Data Modeling** – Organizing data for efficient analysis.

---

## 📊 Dashboard Features

* Interactive charts, slicers, and filters for dynamic analysis.
* Addiction Score vs Mental Health Score analysis using scatter plots.
* Academic level-wise comparison of social media addiction.
* Relationship conflict analysis based on student status.
* Individual student profile analysis using drill-through pages.
* User-friendly navigation using buttons and bookmarks.

---

## 📈 Key Insights

* Social media addiction patterns vary among students with different backgrounds and academic levels.
* The dashboard helps explore the relationship between addiction levels and mental health indicators.
* Social media usage can influence relationship conflicts and behavioral patterns.
* Interactive visualizations make it easier to discover individual student-level insights.
